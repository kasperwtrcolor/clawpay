# SOCIAL_PULSE

Scans for high-sentiment social interactions and attributes rewards to users based on agentic value discovery.

## Metadata
- **ID**: `social_pulse`
- **Name**: SOCIAL_PULSE
- **Type**: Discovery / Attribution

## Configuration
- `min_sentiment`: Minimum sentiment score (0-1) to trigger attribution. Default: `0.8`
- `reward_amount`: Amount of USDC to attribute. Default: `5`
- `keywords`: Array of keywords to track.

## Purpose
The `SOCIAL_PULSE` skill enables the agent to autonomously identify and reward valuable social flow. It parses social mentions and sentiment to distribute the Agent Treasury to the decentralized swarm.
